\section{Data Acquisition}
\label{sec:theory}

    The Country Timeseries datasets were downloaded as excel files from the WITS website. The International Trade Indicators interface available on the database was accessed. Upon loading, the Trade Indicators (variables listed above) were accessed individually through the website's integrated "Search For Indicator" search bar. The selected indicators were Exports of goods and services (\% of GDP), Imports of goods and services (\% of GDP), GDP (constant 2010 USD), Taxes on goods and services (current LCU), Taxes on international trade (current LCU), Customs and other import duties (\% of tax revenue) and Imports of Goods and Services (BoP current USD).
     

\begin{itemize}

\item The international trade indicators were downloaded as individual excel files for each correspondingly: 7 excel files were downloaded and saved listing the complete WITS indicator observations recorded for 193 countries across 1988-2022.

\item The 7 excel files were each imported into Stata and saved individually as a dta file. All data files were merged using Stata's append command and combined into one dataset.

\end{itemize}

\section{Data Manipulation}
\label{sec:data}
    
\begin{itemize}

\item Removing the years not included in 2001-2020: The variables "C, D, E, F, G, H, I, J, K, L, M, N, O, AJ, AK" were removed using Stata's drop command.

\item Removing the additional countries included: The "CountryName" observations were manipulated to only keep the countries of "Australia, France, Israel, Korea, Rep., Norway, Romania, Switzerland, and the United States" using Stata's keep if command.

\item Renaming letter variables: The variables "P, Q, R, S, T, U, V, W, X, Y, Z, AA, AB, AC, AD, AE, AF, AG, AH, AI" were manipulated to be labeled as "yr01, yr02, yr03, yr04, yr05, yr06, yr07, yr08, yr09, yr010, yr 011, yr012, yr013, yr014, yr015, yr016, yr017, yr018, yr019, yr020" using Stata's rename command.

\item Transforming "year" into a singular variable observing the years 2001-2020: The variables observing years were reshaped "long yr0" and "yr0" was renamed "value" before using Stata's replace command to replace "year" values with "year + 200".

\item Reshaping individual countries as variables: The variable "CountryName" was encoded using Stata's encode and generate command to generate a variable labeled "CountryID". The "CountryName" variable was dropped and Stata's reshape wide command was used to reshape "year" and "IndicatorName" as i and "CountryID" as j.

\item Renaming value variables: The variables "value1, value2, value3, value4, value5, value6, value7, value8" were manipulated to be labeled as "Australia, France, Israel, Korea, Rep., Norway, Romania, Switzerland, USA".

\end{itemize}
